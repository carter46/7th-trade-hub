# PWA Icons (required for production)

Add these files to this folder for the PWA to work correctly on install and home screen:

- **icon-192x192.png** (192×192 px)
- **icon-512x512.png** (512×512 px)

Optional: `icon-maskable.png` (512×512, maskable for Android).

The manifest references `/icons/icon-512x512.png` and `/icons/icon-192x192.png`. Without them, the app still runs but install prompts may show a generic icon.
